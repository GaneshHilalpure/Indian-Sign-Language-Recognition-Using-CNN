# Indian-Sign-Language-Recognition-Using-CNN
Indian Sign Language Recognition using CNN to classify hand gestures for assistive communication

## 📌 Project Overview

This project focuses on recognizing **Indian Sign Language (ISL)** gestures using deep learning. The model is trained on hand gesture images and predicts the corresponding sign, helping bridge communication between hearing-impaired and non-sign language users.

---

## 🎯 Objectives

* Recognize ISL hand gestures from images
* Classify different signs accurately
* Build a CNN-based deep learning model
* Enable assistive communication using AI

---

## 📂 Dataset Description

The dataset contains labeled images of hand gestures representing different ISL alphabets or words.

### Features:

* Image data of hand signs
* Multiple gesture classes
* Variations in lighting and background

---

## 🧹 Data Preprocessing

* Resize images to fixed size
* Normalize pixel values
* Convert to arrays
* Train-test split

---

## 📊 Exploratory Data Analysis

* Visualized gesture samples
* Checked class balance
* Observed variations in hand shapes

---

## 🧠 Model Used

Convolutional Neural Network (CNN)

### Why CNN?

* Best for image recognition
* Extracts spatial features
* High performance in classification
